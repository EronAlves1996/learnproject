Controle de Herança

O CSS é uma linguagem da web realmente incrível, porém, apesar de parecer simples, ela é mais complexa do que parece
Existem algoritmos específicos que controlam os estilos impostos aos documentos que evitam haja conflito entre regras.
Os algoritmos escolhem qual regra tem precedência sobre a outra a partir dos seguintes critérios:

  • Especificidade
  • Ordem na cascata
  • Importância
